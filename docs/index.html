04 · Safety perimeter

Quiet until consequence matters.

The hook contract is intentionally small. It protects secrets and repository-changing commands without turning every shell command into a conversation.

ASK

Secrets and environment

Credential files, private keys, `.env` data, secret variables, and likely exfiltration paths.

ASK

Removal

Every `rm`, including wrapped and nested shell forms.

ASK

Repository publication

`git commit`, `git commit-tree`, and `git push` always require approval.

GO

Ordinary development

Builds, tests, formatters, package managers, edits, and read-only Git continue automatically.

Codex needs a push→returns exact command→Claude reviews it→CAS asks you once
